## Cryptocurrency Tracking And Analysis System

### Capstone Project 1 - Source Handle Crawled Datas

### How to run this project?

#### 1. Clone project

```bash
$ git clone https://github.com/Capstone-C1SE04-TI/handle-crawled-datas.git
```

#### 2. Change directory to folder handle-crawled-datas

```bash
$ cd handle-crawled-datas
```

#### 3. Install dependencies

```bash
$ npm install
```

#### 4. Run in developement mode

```bash
$ npm start
```
